## Description

- **`database/`**: Contains database operations such as creating users, checking balances, and handling transactions.
- **`src/`**: Includes the core payment system functionality like creating accounts, transferring funds, and viewing transaction history.
- **`tests/`**: Contains unit tests for the payment system functionality.
- **`README.md`**: Provides an overview of the project, how to run it, and other relevant information.
- **`requirements.txt`**: Specifies the Python packages required for the project.
- **`run.py`**: The main entry point for the user to interact with the payment system via the command line.
